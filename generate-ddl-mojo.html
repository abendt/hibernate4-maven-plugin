hibernate4:generate-ddl

Full name:

de.akquinet.maven.hibernate4:hibernate4-maven-plugin:0.2:generate-ddl

Description:

use Hibernate SchemaExport to generate DDL drop/create scripts.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: compile.
  • Binds by default to the lifecycle phase: prepare-package.

Required Parameters

Name Type Since Description
persistenceUnitName String - Name of the Persistence Unit that should be processed.
User property is: hibernate4.persistenceUnitName.

Optional Parameters

Name Type Since Description
createScriptFileName String - The name of the file to which to write the create script.
Default value is: create.sql.
User property is: hibernate4.createFilename.
delimiter String - Set the end of statement delimiter.
Default value is: ;.
User property is: hibernate4.delimiter.
dropScriptFileName String - The name of the file to which to write the drop script.
Default value is: drop.sql.
User property is: hibernate4.dropFilename.
encoding String - the encoding used to write the files.
Default value is: UTF-8.
User property is: hibernate4.encoding.
generateCreateScript boolean - generate the create script.
Default value is: true.
User property is: hibernate4.generateCreateDdl.
generateDropScript boolean - generate the drop script.
Default value is: true.
User property is: hibernate4.generateDropDdl.
outputDirectory File - Specify where to place generated source files created by schema export.
Default value is: ${project.build.directory}/generated-resources/ddl.
User property is: hibernate4.outputDirectory.
